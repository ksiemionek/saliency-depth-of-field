# Eye tracking

## Setup

1. Clone TranSalNet repository

```bash
   git clone https://github.com/LJOVO/TranSalNet.git
```

2. Download pretrained models [TranSalNet repository](https://github.com/LJOVO/TranSalNet) and place them in `pretrained_models/`:

   - `densenet161-8d451a50.pth`
   - `resnet50-0676ba61.pth`
   - `TranSalNet_Dense.pth`
   - `TranSalNet_Res.pth`
